# Onboarding a joiner
### Transworld PeopleOps Portal — a guide for the team

This is part of the portal guide. It's written to be read alongside the app —
press the **?** on any page for the same help in a panel. The portal is a
**control room**: it records and checks pay, bonus and compliance evidence;
HumanManager and Remita still pay people. Figures shown in the portal are
provisional records in Naira (₦).

---
### Onboarding

Tracks new joiners through their first weeks — the task checklist, probation milestone and first review. It picks up where hiring hands off and makes sure nothing is missed before a person is fully settled.

**Who can use this:** Viewable by several roles; People Ops runs it (onboarding.manage).

**What you can do here:**

- **Start onboarding** — Begins the onboarding record for a joiner.
- **Open a joiner** — Opens their onboarding detail and tasks.

**Where this sits:** Hiring → onboarding (tasks + probation + first review) → confirmed employee. Probation outcomes feed the staff file.

**Good to know:**

- Onboarding usually starts from a converted candidate, so the person already exists in Employees.

---

### Joiner onboarding

One joiner's onboarding: the task list (seed a default set or add your own), the probation window, and the scheduled first review.

**Who can use this:** People Ops (onboarding.manage).

**What you can do here:**

- **Seed default tasks** — Adds the standard onboarding checklist.
- **Add / complete tasks** — Adds tasks and marks them done as the joiner settles in.
- **Set probation** — Records the probation period and its milestone dates.
- **Schedule review** — Books the first formal review.

**Good to know:**

- Probation and review dates set here are what the alert and staff-file drives will look for.
